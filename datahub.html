Field Definitions

Field Name Description
Braap Internal Part ID Internal tracking identifier.
Part Name Manufacturer part name.
Description Manufacturer long part description.
Manufacturer Part Number Part number provided by the manufacturer.
Size Defines the assorted sizes of a part. If the size changes the price of the part, create another part. Braap’s AI engine shows the customer one part with multiple options when searching.
Color Defines the assorted colors of the part. If the color changes the price of the part, create another part .Braap’s AI engine shows the customer one part with multiple options when searching.
Part Condition Defines the state of the part: New or Used - Not new, pre-owned.
Braap Master System The AI classification set by Braap for search and results management.
Braap Subsystem The AI classification set by Braap for search and results management.
Part Origin Defines the source of the part: OEM-Original part from the manufacturer | AFT-Aftermarket part | Universal – Fits a wide range of machines or assets.
Supplier System Each supplier classifies parts into systems so they can be found. Every supplier uses a different method of classifying parts. For example, a brake pad for a vehicle is part of the brake system, so the supplier will classify it as "Brake System". The supplier system is provided by the supplier of the part
Sale Price The part price that is displayed to customers and technicians. It is the supplier price plus a markup. The price can be shown in any currency.
Display Currency Represents the currency of the price of the part.
Supplier Name Legal name of the supplier. Do not include LLC, Inc, or Corp.
Order Placement Email Address The email address used to place orders if electronic order placement is not available.
Supplier Price The price your company buys the part for from the supplier.
Sale Price Markup or Discount Percentage The percentage your company increases the supplier price to the customer. When this field the Sale Price field is automatically calculated
Show as Discount from Sales Price Sometimes suppliers provide a discount from the Sale Price. The discount can be provided as a factor or a percentage. If it is provided as a factor convert it to as percentage. When this field is used Supplier Price is automatically calculated.
Street Address Address where the part order is fulfilled. If there is a warehouse associated with the supplier, then the Supplier address is their head office location.
Address Line 2 Use this field to define a building name, floor, suite, or additional address details.
City, State, Zip / Postal Code Supplier address details.
API Connection no Download of Parts Some suppliers require integration through their API and do not provide an initial load of their parts. In this case, we setup a part template and add the parts to the database as they are needed.
Order Placemenet Method Some suppliers place orders electronically through an API, while others require a manual order. For manual orders Braap sends the order details automatically to the supplier order placement email address.
Provide Supplier Box for Shipping Some suppliers require the customer facing company to send them a box and pre-paid label for shipping.
Send Returns to Supplier If this box is selected part returns will be sent to the supplier location.
Measuring Units Defines the units used in part weight and size. Imperial measurement is inches and pounds. Metric is kilograms and centimeters.
WareHouse
Email Address The email address used to place orders if electronic fulfillment notification is not available.
Warehouse Address The address where supplier fulfillment takes place.
Address Line 2 Use this field to define a building name, floor, suite, or additional address details.
City, State, Zip / Postal Code Supplier address details.
Status A part can have several statuses that are set by a change in the supplier inventory levels. Note, a part is In Stock if the inventory level is more than 1.
Inventory Level The number of parts at the warehouse. If the value is blank, then Braap does not automatically set the Status field. If the inventory level is greater than 1 and the last part is sold, Braap updates the Status to Out of Stock.
Expected Ship Time Defines the time from order placement to supplier shipment creation.
Delivery Mode Suppliers that ship on your behalf to the customer are set to Customer Direct. Suppliers that ship to you and then you ship to the customer are set as Dropship.
Weight The weight of the part in imperial or metric value. This field is controlled by the Measuring Units field.
Length The length of the part in imperial or metric value. This field is controlled by the Measuring Units field.
Width The width of the part in imperial or metric value. This field is controlled by the Measuring Units field.
Height The height of the part in imperial or metric value. This field is controlled by the Measuring Units field.
Contains Battery Shipping companies need to know if the part contains a battery. Set to yes if the part contains a battery.
Fluids Shipping companies need to know if the part contains fluids or is a fluid. Set to Yes if the part contains fluids or is a fluid.
Rating Rating from 1-5 for user feedback about the part.
Review Description of a user experience with the part.
Fitments Defines the actual machine or asset the part is known to fit. It is critical to include all fitment possibilities, so technicians and customers are not confused or misled when evaluating and ordering parts.
Industry The market segment that the part relates to. For example, Powersport or Heating & Cooling.
Type The main grouping of machines or assets in that industry. For example, ATV or Furnace.
Make The brand of a machine or asset. For example, ATV brand is Polaris or for a Furnace Trane.
Model The brand’s product of a machine or asset. For example, Polaris ATV the model is Sportsman 570 or for a Trane furnace the model is S9V2.
Horsepower Used for machines or assets with fitments that require defined horsepower. An example are boats.
Related Parts Recommendations that complement or enhance the item a customer is currently viewing.
Warranty Period (Number of Days from Purchase or Commission Date) The number of days the supplier either on their own or through the manufacturer provide a warranty on the part. Depending on the supplier or manufacturer policy, the warranty can be from date of purchase or part commission.
Track Warranty from Commission Date When this field is used the warranty period is calculated from the date the part is commissioned.
Warnings or Notices Products that are regulated or sold in specific areas sometimes are required to provide disclosures for customer protection.
Recall Date The date of the recall.
Recall Number The number provided by the recall agency or company.
Link to Recall Notice Provide hyperlinks to recall notices.
Box Size The Braap AI engine creates the best box size based on shipping company typical box sizes.
Container Name The name assigned to the shipping package or box by the carrier.
